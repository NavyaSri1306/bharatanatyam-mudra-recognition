# Bharatanatyam Mudra Recognition

A deep learning based Bharatanatyam hand gesture recognition system developed using ResNet50 for identifying Samyukta and Asamyukta mudras.

## Features
- Real-time mudra prediction
- Image classification using ResNet50
- Data augmentation and preprocessing
- Accurate gesture recognition

## Technologies Used
- Python
- TensorFlow
- Keras
- OpenCV
- NumPy
